# Клиент-серверное приложение "Телефонный справочник"

## Запуск

1. Используется сторонняя библиотека `jsoncpp`.  
Установка: `sudo apt-get install libjsoncpp-dev`

2. Сборка проекта: `make`
3. Запуск сервера: `./server`
4. Запуск клиента: `python3 client.py`

## Функционал 
Приложение поддерживает базовый функционал:
1. Добавление записи в телефонную книгу
2. Удаление записи по номеру телефона
3. Поиск записи по любому полю (кроме заметки)
4. Вывод всех записей

## Зоны роста/нужно доделать, но не успел
1. Провести рефакторинг кода. Переписать в ООП стиле.  
   На данный момент не совсем аккуратно ведется работа с ресурсами. 
   Из-за этого могут возникнуть проблемы с очисткой ресурсов. Так же из-за
   этого приложение сложно масштабировать. Необходимо аккуратно переписать
   под идиому `RAII`.
2. Создавать отдельный поток для каждого запроса не самая правильная идея.
Такой подход может привести к голоданию. Лучше в данной задаче использовать
`Thread Pool`. Он поможет избежать данной проблемы.
3. Добавить проверки на входные данные/корректность ввода данных. 
На данный момент никаких проверок на корректность данных нет, но они необходимы.
